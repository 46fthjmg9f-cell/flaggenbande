Flaggenbande quiz-video Production Engine without changing the existing iOS app
or analytics dashboard.

## 7. Rendered example

Not applicable to 0.1.0. The approved release plan places the renderer and first
example video in 0.3.0, after a validated country-data contract is available in
0.2.0. Installing Remotion or fabricating flag data now would violate that
sequence. Startup, doctor, tests and builds are the release evidence for this
foundation.

## 8. Known issues

- the app catalog is not yet safe as a production database
- remote FlagCDN/Wikimedia URLs are not approved local production assets
- several codes require policy or correction, including the conflicting `NC`
  entry and non-ISO subdivision/custom codes
- capital localization, sources, licenses, checksums and review metadata are
  missing
- the existing dashboard bundle reports a non-blocking large-chunk warning
- local verification used Node 26.5.0; CI and `.node-version` verify the required
  Node 22 baseline

## 9. Manual steps

No credential, cloud or paid-service setup is required. For a fresh checkout,
install Node 22 and run:

```bash
cd /Users/praemer/Projects/flaggenbande-content-r010/content-system
npm ci
npm run doctor
npm run check
npm run build
```
