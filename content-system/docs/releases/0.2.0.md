# Release 0.2.0 — Country Database

Status: **in development; human data approval pending**

## Scope

- deterministic import of the 193 UN-member-state seed
- pinned CLDR names and geographic metadata
- dated CC0 capital snapshot
- 193 local MIT-licensed SVG flags
- strict schema, provenance, checksums and review queue
- offline validation and reproducibility tests
- immutable OneDrive candidate package
- exact-set gates for both UN M49 and Wikidata snapshots
- complete review, difficulty, flag, provenance and license release gate
- two-pass reproducibility plus post-copy SHA-256 verification

## Explicit non-scope

- territories, observer states and disputed entities
- automatic difficulty learning
- video rendering
- platform upload or performance collection

## Approval gate

Before the dataset can be tagged as stable, a human must approve the scope,
name/geography exceptions, capital-role review, initial difficulty assignments
and the visual flag contact sheet. Until then every generated record remains a
candidate and is blocked from production.

`remoteSyncVerified` remains false until OneDrive visibly confirms that the
mounted candidate directory has reached Microsoft's cloud. A successful local
mount re-read is necessary but deliberately not reported as cloud confirmation.
