# Flaggenbande

Dieses Repository enthält die bestehende iOS-App und, getrennt davon, das
Fundament für die automatisierte Produktion kurzer Flaggen-Quizvideos.

## Bereiche

- `SpassmitFlaggen/`: bestehende iOS-App „Flaggenbande“
- `dashboard/`: bestehendes, unabhängiges App-Monitoring
- `content-system/`: neue stabile Production Engine für Quizvideos

Die Content-Produktion übernimmt keine App- oder Dashboard-Logik. Einstieg,
Installation und Betriebsbefehle stehen in
[`content-system/README.md`](content-system/README.md).
